## Get Started

After installation, start the Ollama desktop app and open a new chat.

Choose a model, sign in when prompted, and begin chatting. The desktop app also exposes the local API at `http://localhost:11434`.

## API

Use the [API](/api) to integrate Ollama into your applications:

```sh
curl http://localhost:11434/api/chat -d '{
  "model": "gemma3",
  "messages": [{ "role": "user", "content": "Hello!" }]
}'
```

See the [API documentation](/api) for Python, JavaScript, and other integrations.
